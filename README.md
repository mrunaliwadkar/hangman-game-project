# 🎯 Hangman Game

A fun and interactive Hangman game built with Python and Tkinter. Guess the hidden word letter by letter before running out of attempts!

## Features

✨ **Interactive GUI** - Clean and user-friendly graphical interface
🎨 **Colorful Design** - Modern UI with color-coded feedback
💡 **Hint System** - Get up to 2 hints per game
📝 **Word Categories** - Wide variety of programming and tech-related words
🔄 **Easy Restart** - Start a new game anytime
💻 **ASCII Art Hangman** - Visual representation of the game state

## Requirements

- Python 3.x
- Tkinter (usually comes with Python)

## Installation

1. Clone this repository or download the `hangman.py` file:
```bash
git clone https://github.com/mrunaliwadkar/hangman-game-project.git
cd hangman-game
```

2. Ensure you have Python 3.x installed on your system:
```bash
python --version
```

## How to Run

Simply execute the script with Python:

```bash
python hangman.py
```

The game window will open, and you're ready to play!

## How to Play

1. **Guess Letters** - Enter a single letter in the text field and click the "Guess" button
2. **Track Progress** - See the hangman drawing progress and guessed letters
3. **Use Hints** - Click the "💡 Get Hint" button up to 2 times for clues about the word
4. **Win** - Complete the word before the hangman is fully drawn
5. **Lose** - If the hangman is completed (6 wrong guesses), the word is revealed
6. **Restart** - Click "🔄 Restart" to start a new game

## Game Rules

- You have 6 chances to guess wrong letters
- Each correct letter reveals its position(s) in the word
- You can use up to 2 hints per game
- The game ends when you either complete the word or reach 6 wrong guesses
- Words are randomly selected from a list of tech-related terms

You can easily customize the game by modifying:

- **Word List**: Edit the `words` list in the code
- **Hints**: Modify the `word_hints` dictionary

- **Difficulty**: Change `max_hints` value for easier/harder gameplay
